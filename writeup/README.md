# Write-up

A short technical write-up of this project's findings, in LaTeX
(`main.tex`, plain `article` class — no external style files needed beyond a
standard TeX distribution). The compiled `main.pdf` is included.

## Build

```bash
cd writeup
tectonic main.tex                 # one command: auto-fetches packages + runs BibTeX
# or, with a full TeX Live install:
pdflatex main.tex && bibtex main && pdflatex main.tex && pdflatex main.tex
# or: latexmk -pdf main.tex
```

Or upload `writeup/` to a LaTeX editor of your choice.
